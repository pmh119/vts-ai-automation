ℹ️ RPA 구동을 위한 WebView2 런타임 설치 가이드

🔒 내부망 및 폐쇄망 PC를 위한 WebView2 오프라인 설치 및 DLL 빌드 환경 구성법

💡 오프라인 DLL 배치가 필수인 이유:
PowerShell RPA 자동화 스크립트 실행 시 WinForms GUI와 Chromium 브라우저를 연동하기 위해서는 WebView2 핵심 동적 라이브러리(DLL) 모듈이 로컬 특정 경로에 정확하게 탑재되어 있어야 합니다.
  • 1. 런타임(Standalone) 다운로드 및 보안망 이동
    인터넷이 연결된 외부 PC에서 Microsoft 공식 WebView2 다운로드 페이지에 접속하여 Evergreen Standalone Installer (x64) 버전을 다운로드한 뒤, 내부 승인 매체(보안 USB 등)를 통해 내부망 관제 PC로 파일(MicrosoftEdgeWebview2Setup.exe)을 안전하게 복사합니다.
  • 2. 명령 프롬프트(CMD)를 통한 강제 오프라인 무소음 설치
    관리자 권한으로 실행한 명령 프롬프트(CMD)에서 설치 파일 경로로 이동 후 아래 명령어를 입력하여 네트워크 통신 없이 로컬 디렉토리에 설치를 마무리합니다: 
    MicrosoftEdgeWebview2Setup.exe /silent /install
  • 3. [핵심] C:\webview2\ 라이브러리 및 DLL 폴더 구조 수동 구성
    관제 스크립트가 참조할 수 있도록 C드라이브 바로 밑에 C:\webview2 폴더를 생성하고, 필수 작동 DLL들을 다음과 같이 완벽한 디렉토리 트리 구조로 배치해야 합니다: 
    C:\
└── webview2\
    ├── Microsoft.Web.WebView2.Core.dll  (RPA Core 인터페이스 API)
    ├── Microsoft.Web.WebView2.WinForms.dll  (윈폼 데이터 시각화 라이브러리)
    └── runtimes\
        └── win-x64\
            └── native\
                └── WebView2Loader.dll   (x64 아키텍처 필수 로더)
    ※ 해당 DLL 파일들은 배포 패키지 내부 `사례3/` 폴더 내부에 사전 동봉되어 있으므로 그대로 지정 경로에 복사해 넣으시면 됩니다.
  • 4. 제어판을 통한 설치 완료 여부 크로스체크
    설치 완료 후 [제어판] -> [프로그램 및 기능]에서 Microsoft Edge WebView2 Runtime 항목이 리스트에 존재하며, 버전 정보(예: 110.x.xxxx.xx 이상)가 정상 표출되는지 검증합니다.
+ 소스코드 전문 공개

실제 구동되는 스크립트 소스코드 원본입니다.

※ 내부망 도메인은 보안을 위해 더미값(hp.kcg.internal)으로 마스킹 처리되어 있습니다.

portmis.ps1
⬇ 다운로드 
# PORT-MIS Auto-Reporter
# WebView2 기반 RPA 자동화 스크립트
Add-Type -AssemblyName System.Windows.Forms
Add-Type -AssemblyName System.Drawing

$wv2Core     = "C:\WebView2\Microsoft.Web.WebView2.Core.dll"
$wv2WinForms = "C:\WebView2\Microsoft.Web.WebView2.WinForms.dll"
Add-Type -Path $wv2Core
Add-Type -Path $wv2WinForms

# 내부망 URL (더미 처리)
$PORTMIS_URL       = "https://portmis.go.kr"
$FIELD_LOGIN_URL   = "http://hp.kcg.internal/FisdOBS/index.jsp"
$FIELD_DIARY_URL   = "http://hp.kcg.internal/FisdOBS/vt/vtsWorkDiary.do?progrm_id=OBS0501"

# GUI Form 초기화
$form = New-Object System.Windows.Forms.Form
$form.Text = "PORT-MIS 관제일보 자동화"
$form.Size = New-Object System.Drawing.Size(900, 650)

# WebView2 컨트롤 초기화
$webView = New-Object Microsoft.Web.WebView2.WinForms.WebView2
$webView.Dock = "Fill"

# 터미널 출력 패널
$terminal = New-Object System.Windows.Forms.RichTextBox
$terminal.BackColor = [System.Drawing.Color]::FromArgb(13,17,23)
$terminal.ForeColor = [System.Drawing.Color]::FromArgb(230,237,243)
$terminal.Font = New-Object System.Drawing.Font("Consolas", 9)

# 실행 버튼
$runBtn = New-Object System.Windows.Forms.Button
$runBtn.Text = "엑셀 및 현장업무포털 실적 입력 실행"
$runBtn.Add_Click({
    Write-Terminal "[시작] PORT-MIS 자동 접속 중..."
    # 웹 크롤링 및 데이터 처리 로직 실행
})
# ... (이하 GUI 조립 및 이벤트 핸들러 생략)
